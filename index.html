Problema numero 3

\[\begin{aligned} \lim_{n \to \infty} \frac{1^2 + 2^2 + ... + n^2}{ (1+n)(2+n)^2 } \end{aligned}\]

Problema numero 3

\[\begin{aligned} \lim_{n \to \infty} \frac{\frac{1^2 + 2^2 + ... + n^2}{n^3}}{\frac{(1+n)(2+n)^2}{n^3} } \end{aligned}\]
Por el teorema de la triple integral
\[\begin{aligned} \lim_{n\to \infty} \frac{1^2 + 2^2 + ... + n^2}{n^3} & = \int_0^1 x^2 dx \\ = \left.x^3\right\rvert_0^1 & = \frac{1}3 \end{aligned}\]

Problema numero 3

\[\begin{aligned} \lim_{n \to \infty} \frac{\frac{1}{3}}{\frac{(1+n)}{n}\frac{(2+n)^2}{n^2} } \end{aligned}\]

Problema numero 3

\[\begin{aligned} \lim_{n \to \infty} \frac{1}{3} \frac{1}{(\frac 1 n +1)(\frac 2 n +1)^2 } \end{aligned}\]

Respuesta

\[\begin{aligned} \lim_{n \to \infty}\frac{1^2 + 2^2 + ... + n^2}{ (1+n)(2+n)^2 } = \frac{1}{3} \end{aligned}\]
